# Contributing

This repository is closed to public contributions. 

## Reporting Issues

If you encounter any issues or have suggestions for improvements, please:

1. Check existing issues to see if the problem has already been reported
2. Create a new issue with:
   - A clear description of the problem or suggestion
   - Steps to reproduce (if applicable)
   - Expected vs actual behavior
   - Environment details (OS, Python version, etc.)

## Security Issues

For security-related issues, please see [SECURITY.md](SECURITY.md) for reporting procedures.

## Code of Conduct

This project follows the Snyk Labs community guidelines. Please be respectful and constructive in all interactions.
